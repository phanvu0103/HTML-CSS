CSS Units

  • .Absolute Units (tuyệt đối: giá trị hiện thị chính xác, cố định)
    • PX (pixel: điểm ảnh)
  • .Relative Units (Tương Đối: phụ thuộc và đối tượng khác )
    • % (Phụ thuộc vào thẻ cha )
    • VW, VH (phụ thuộc vào khung hiển thị trình duyệt)
    • Em (đại diện cho kích thước font chữ)
    • rem (đại diện cho kích thước font chữ)

CSS Flexbox ?

Display: flex | inline-flex:
(quyết định sử dụng layout box hay ko)
bạn muốn tạo ra đc flex container và main start thì sử dụng Display: flex | inline-flex

Flex-direction: row(ngang) | column(dọc) : br mặc định Display main axis là ngang ta có thể thay đổi thành dọc

Flex-wrap:(ko xuống dòng) nowrap | warp(xuống dòng) | warp-reverse (sẽ nhảy lên trên đảo chiều cross end lên cross start )

Flex-basis: xét kích thuốc thằng main size < length >

Justify-content(xét thuộc tính phần cha như flex container thì ko cần xét Justify-self vì Justify-self xét thuộc tính phần con ):(canh phương hướng flex item main axis bên phải trái hay giữa trong trường hơp main axis nằm ngang)
(còn main axis nằm dọc thì canh lên trên , dưới , giữa )
Flex-start | flex-end | center | space-between |space-around

Justify-self(xét thuộc tính phần con < Justify-content > thì thuộc tính Justify-self cho phần con ): flex-start | flex-end | center

Align-content(giống như Justify-content như khác với trục cross axis ko phải trục main axis): flex-start | flex-end | center

Align-sefl(giống như Justify-self chỉ khác xét cho thằng Align-content ): flex-start | flex-end | center

Flex-grow: làm lớn lên nở ra(dùng để tăng kích thước thằng main size trong flex item): < number> kích thước này linh động chia theo trọng số hay tỉ lệ

Flex-shrink: làm thu nhỏ lại(ngược với thằng Flex-grow qua ví dụ mới thấy): < number>

Flex: cú pháp viết ngắn viết gộp lại 3 thằng Flex-basis | Flex-grow | Flex-shrink dùng Flex có thể viết gộp lại 3 thằng như trên < number>

Order: < number>

bình thường thì xuất hiện theo mặc định hay theo điểm main start hay điểm start hay điểm end
nếu bạn can thiệp vào thì dùng lệnh Order để thay đổi vị trí mình mong muốn

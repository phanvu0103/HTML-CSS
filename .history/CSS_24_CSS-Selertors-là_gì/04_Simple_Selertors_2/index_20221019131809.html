Selector Example Example description .class .intro Selects all elements with class="intro" .class1.class2 .name1.name2 Selects all elements with both name1 and name2 set within its class attribute .class1 .class2 .name1 .name2 Selects all elements with name2 that is a descendant of an element with name1 #id #firstname Selects the element with id="firstname" * * Selects all elements element p Selects all <p> elements element.class p.intro Selects all <p> elements with class="intro" element,element div, p Selects all <div> elements and all <p> elements element element div p Selects all <p> elements inside <div> elements element>element div > p Selects all <p> elements where the parent is a <div> element element+element div + p Selects the first <p> element that is placed immediately after <div> elements

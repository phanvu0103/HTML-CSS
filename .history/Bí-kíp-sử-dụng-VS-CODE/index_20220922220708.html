Bí Kíp Sử Dụng TML CSS

  1. __'ctrl + P' : Command Plalette (tìm kiếm)
  2. __'ctrl + G': Go To Line (tìm đường line như dòng 31 củadòng "body")
  3. __'ctrl + D': Find Match (tìm đoạn khớp nhau code (li)cùng sữa đoạn>
  4. __'ctrl + L': Find Match (tìm và chọn đoạn khớp nhau code (li)cùng sữa đoạn>
  5. __'shift + end': Full Line (chọn về cuối dòng)
  6. __'shift + home':Full Line (chọn về đầu dòng)
  7. __'Alt + click': Multiple Cursor (giống ctrl D nhưng mình có thể click những dòng khác nhau theo ý muốn và sữa nội dung)
  8. __'Alt + Arrow up down': Move up move dơwn (chuyển 1 dòng hay nhiều dòng lên xg theo ý muốn)
  9. __'Alt + shift + Arrow up down'>: Fast copy same line (copy nhanh 1 dòng hay nhiều dòng)

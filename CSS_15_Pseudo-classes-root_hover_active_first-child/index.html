CSS Pseudo Classess : Lớp giả

  • :root đã học biến css phần tử gốc html
  • :hover dịnh nghĩa vô số css,chỉ kích hoạt chuột vào đối tượng đó VD: h1
  • :active bấm chuọt vào đối tượng sẽ đổi màu VD box (box:active)
  • :first-child nghĩa là đứa con đầu tiên hay dòng đầu tiền VD li:root
  • :last-child nghĩa là đứa con cuối cùng hay dòng cuối cùng VD li:last-child

    So sánh display: none , opacity, Visibility
  • display: none.
  • opacity
  • Visibility (hiện lên)

Trong bài này chúng ta sẽ so sánh giống nhau và khác nhau giữa các thuộc tính CSS sau:

  • display: none;
  • opacity: 0;
  • visibility: hidden;

Là người mới, chúng ta thường khó phân biệt được khác nhau trong trường hợp này. Vì vậy, đây cũng là một câu hỏi thường được sử dụng khi đi phỏng vấn với vai trò Thực tập sinh Frontend, Fresher Frontend, v.v.

Giống và khác nhau

# display: none opacity: 0 visibility: hidden

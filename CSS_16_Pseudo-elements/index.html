CSS Pseudo Elements : Phần tử giả
Dòng thứ 2

  • :before là phía trước VD: taọ hộp box nhỏ màu xanh trên hộp box lớn chỉ tạo đc 1 before nó chỉ tồn tại là 1 ko có 2 trên box thôi
  • :after là phía sau VD: taọ hộp box nhỏ màu đỏ trên hộp box lớn chỉ tạo đc 1 after nó chỉ tồn tại là 1 ko có 2 trên box thôi
  • :first-letter ký tự đầu tiên to hay nhỏ trên H1 CSS chữ C to lên
  • :first-line dòng đầu tiên trên h1 thay đổi cỡ chữ màu sắc mình tạo
  • :selection bôi đen trên dòng chữ h1 CSS nó sẽ hiện màu sắc hay background-color mình muốn

# Tiêu chí chọn font chữ cho Developer

  1. Monospace: tất cả các ký tự và khoảng trắng đều cùng một cỡ, điều này cho phép canh cột và lề dễ dàng và code dễ đọc hơn với những đoạn logic lặp lại. Các font mặc định cho code editor và terminal đều là monospace.

  2. Dễ dàng phân biệt các ký tự Il1i, 0Oo, CG: có nhiều font monospace hiển thị các ký tự kể trên rất giống nhau, khiến cho việc đọc code khó khăn hơn. Với tiêu chí này, tôi đã loại một số font monospace khá phổ biến như Courier New, Noto Mono, Droid Sans Mono…

  3. Rõ, dễ đọc các ký tự đặc biệt cho cú pháp phải: () [] {} =+-<>:;, ."'

  4. Hỗ trợ tiếng Việt Unicode để hiển thị các chuỗi văn bản hoặc ghi chú bằng tiếng Việt (nếu có). Tiêu chí này cũng loại khá nhiều font mono đẹp và phổ biến khác như: Ubuntu Mono, Menlo, Operator Mono, v.v.

Các font chữ mình đề cập trong bài này hầu hết đều đáp ứng các tiêu chí trên, có thể có một số ít font chưa hỗ trợ tốt Tiếng Việt. Nếu gặp font chưa hiển thị tốt tiếng Việt, bạn hãy sử dụng thêm một font phía sau làm dự phòng nhé.

# Font chữ phổ biến cho VS Code

1. Consolas

Font này không miễn phí nhưng có sẵn trong Windows hoặc được cài theo bộ Microsoft Office.

2. Monaco

Font này có sẵn trên macOS.

3. Roboto Mono

Font monospace của bộ Roboto, font mặc định cho HĐH Android và material design.

4. Source Code Pro

Font monospace được dựng bởi Adobe dành riêng cho việc hiển thị code nên rất dễ đọc và tròn trịa.

5. Space Mono

Font monospace được dựng bởi Colophon Foundry cho Google Fonts. Font này có dáng chữ khá lạ và mang hơi hướng mỹ thuật hơn là cho code.

# Font chữ hỗ trợ ligatures

Ligature: đây là hiện tượng nối nhiều chữ thành một xuất hiện cùng với kỹ thuật in ấn. Ngày nay, ligature được mô phỏng lại (chủ yếu là vì lý do mỹ thuật) thông qua một số font chữ có hỗ trợ.

1. SF Mono

Font SF Mono dùng để hiển thị code bên trong XCode 8+.

2. Jetbrains Mono

Font chữ của Jetbrains - thương hiệu nổi tiếng của các IDE đình đám như PhpStorm, WebStorm, v.v.

3. Operator Mono

Font chữ khá là nghệ thuật, bay bướm, có phiên bản hỗ trợ ligatures.

4. Fira Code

Font chữ quá nổi tiếng có hỗ trợ ligatures.

5. Cascadia Code

Font được thiết kế bởi Microsoft cho ứng dụng terminal mới, khá bay bướm & hỗ trợ tốt tiếng Việt.

6. Victor Mono

Font chữ dành cho các coder yêu thích “nghệ thuật”, rất bay bướm, tuy nhiên nét hơi mảnh.

7. Hasklig

Font chữ phát triển từ Source Code Pro bằng cách bổ sung thêm ligatures.

Một số fonts khác có hỗ trợ ligatures: https://github.com/tonsky/FiraCode#alternatives

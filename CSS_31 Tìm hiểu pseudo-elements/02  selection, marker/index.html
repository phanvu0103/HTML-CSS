CSS - The ::marker Pseudo-element (

Phần tử giả ::marker chọn các điểm đánh dấu của các mục trong danh sách.)

The ::marker pseudo-element selects the markers of list items.

Ví dụ sau tạo kiểu cho các điểm đánh dấu của các mục trong danh sách:

Example

::marker {

color: red;

font-size: 23px;

}

CSS - The ::selection Pseudo-element

(CSS - Phần tử giả ::selection)

The ::selection pseudo-element matches the portion of an element that is selected by a user.

(Phần tử giả ::selection khớp với phần của phần tử được người dùng chọn.)

The following CSS properties can be applied to ::selection : color , background , cursor , and outline .

Các thuộc tính CSS sau có thể được áp dụng cho ::selection: màu, nền, con trỏ và đường viền.

The following example makes the selected text red on a yellow background:

Example

::selection {

color: red;

background: yellow

}
